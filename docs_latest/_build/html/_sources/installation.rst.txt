Installation Guide
================

Requirements
-----------
- Python 3.9 or higher
- Git
- pip (Python package installer)

Steps
-----

1. Clone the repository:

   .. code-block:: bash

      git clone <repository-url>
      cd project_management_dashboard

2. Run the setup script:

   .. code-block:: bash

      ./scripts/setup.sh

3. Verify the installation:

   .. code-block:: bash

      ./scripts/lint_and_test.sh
